# ev3dev Rubiks Cube Solver

## Setting project environment

    py -3 -m venv .venv
    .venv\Scripts\activate
    python -m pip install --upgrade pip
    pip install python-ev3dev2

## Input and Output Ports
| Port | Function |
|-|-|
| 1 | Color Sensor |
| 2 | IR Sensor |

| Port | Function |
|-|-|
| A | Flip Arm Motor |
| B | Rotating Platform Motor |
| C | Color Sensor Motor |
